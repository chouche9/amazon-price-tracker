# Amazon Price Tracker

This simple application tracks your favourite prodcuts on amazon and sends you an email when their price falls below your threshold.
The products can be scanned altogether without having to run the code individually.
The search history from past searches along with the current search will be saved for inspection.


# Getting Started
Enter the url of the products that you wish to track, along with their title and price threshold, in the file found at trackers/products.csv.
Change the email and password within Amazon_Scraper.py to send the alert email to your desired account.
Each search results along with the date they were completed will be saved inside the folder search_history.
